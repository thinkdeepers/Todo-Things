# Todo-Things

由Flabe5全权编程撰写，纯本地运行，无上传，无网络需求

简洁，好用，推荐

在intall分支中含有打包好的下载包

https://github.com/thinkdeepers/Todo-Things/tree/install

1、软件能够置顶至软件窗口最顶上，能够自有调整大小，方便放置电脑上
2、底部直接输入可以新建待办
3、可进行计划分类，不同的计划显示不同待办，同时可显示一键所有
3、可选不同主题，深色，浅色，多彩色
4、待办中可以创建子待办
5、待办可设置优先级
6、待办事项可设置时间和是否提醒
7、可编辑是否显示已完成待办

开源地址：https://github.com/thinkdeepers/Todo-Things
